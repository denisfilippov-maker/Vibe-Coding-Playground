# PROJECT_OVERVIEW.md — Vibe Coding Playground

## Что это за проект

**Vibe Coding Playground** — одностраничный showcase-сайт, демонстрирующий возможности современной веб-разработки с помощью AI-инструментов (Claude + Cursor). Каждый блок сайта — живая демонстрация одной мощной техники: анимации, интерактивность, визуальные эффекты, адаптивность.

## Цель

Вдохновить аудиторию и показать, что профессиональный, визуально впечатляющий сайт можно создать за 1 день с помощью вайбкодинга.

## Целевая аудитория

- Люди, интересующиеся вайбкодингом
- Студенты курса по вайбкодингу
- Новички, которые хотят понять мощь AI-инструментов
- Опытные разработчики, ищущие вдохновение

## Ключевые требования

- **Визуально впечатляющий** — каждый блок должен вызывать "вау"
- **Интерактивный** — сайт должен быть живым, реагировать на пользователя
- **Тёмная тема по умолчанию** с возможностью переключения
- **Полностью адаптивный** — работает на всех устройствах
- **Быстрый** — анимации плавные (60fps), нет лагов

## Секции сайта

| # | Секция | Демонстрирует |
|---|--------|--------------|
| 1 | Hero | Градиентные анимации, parallax, stagger reveal |
| 2 | Animations | Framer Motion, scroll-triggered, stagger cards |
| 3 | Interactivity | Конфетти, морфинг кнопок, слайдеры |
| 4 | Visual Effects | Glassmorphism, glow, неон, градиенты |
| 5 | Responsive Design | Интерактивный device-переключатель |
| 6 | Footer | CTA, ссылки, информация о проекте |

## Метрики успеха

- Lighthouse Performance: > 85
- Lighthouse Accessibility: > 90
- Все анимации работают на 60fps
- Корректно отображается на iOS Safari, Chrome, Firefox
